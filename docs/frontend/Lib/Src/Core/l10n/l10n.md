# Localization Barrel File (`l10n.dart`)

## Overview

This module serves as the centralized entry point for localization within the HealthBank frontend application.

It re-exports:

- `AppLocalizations` (generated from ARB files)
- The `BuildContext` localization extension (`context.l10n`)

File: `frontend/lib/src/core/l10n/l10n.dart`

By importing this file, widgets gain access to all localization utilities without needing multiple imports.

---

## Architecture / Design

### Barrel Export Pattern

This file implements the barrel pattern to simplify localization imports.

Instead of importing:

```dart
import 'generated/app_localizations.dart';
import 'l10n_extension.dart';
