# API Services Barrel File (`services.dart`)

## Overview
This file serves as a **barrel export** for all Retrofit API service interfaces used in the frontend application.

By importing this single file, consumers gain access to all defined API services without needing to import each service individually.

File: `frontend/lib/core/api/services/services.dart`

Example usage:

```dart
import 'package:frontend/src/core/api/services/services.dart';
