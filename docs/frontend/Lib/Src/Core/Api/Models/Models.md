# API Models Barrel File (`models.dart`)

## Overview
This file serves as a **barrel export** for all API model definitions used throughout the frontend application.

By importing this single file, consumers gain access to every typed API model without needing to import individual model files manually.

File: `frontend/lib/core/api/models/models.dart`

Example usage:

```dart
import 'package:frontend/src/core/api/models/models.dart';
